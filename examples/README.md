# Examples

Small, runnable snippets using the `@mycl/hypha` SDK.

| File | Purpose |
| --- | --- |
| `publish-price-feed.ts` | Publish a compressed SOL/USD price batch. |
| `subscribe-bot.ts` | Subscribe to a feed and log each tick. |
| `verify-proof.ts` | Verify a Merkle proof envelope returned by the RPC. |

Run with `npx ts-node examples/<file>` after installing dev dependencies.
